Frontend Developer
------------------

As a frontend developer, I focus on crafting visually appealing and intuitive user interfaces for websites and applications. Using my skills in Figma, React, TypeScript, Tailwind CSS, Framer Motion, and more, I bring web designs to life in the browser, ensuring that the site looks great and functions smoothly on all devices. Though challenging at times, I find frontend development rewarding because I can see the visual results of my efforts right on the screen, helping to improve the user's experience.
